# Python Util Scripts

A collection of handy Python utility scripts that I need sometimes.

## Usage

Scripts may explain their usage via `py <script.py> --help` (assuming I implement it).
If not then inspect the source for proper usage.

## Scripts

### `scrape.py` - Web Image Scraper

Download all images from a given URL:

```bash
python scrape.py https://example.com/
```

Images are saved to the `output/` directory.

---

### `rng-words.py` - Random Word Generator

Generate a list of random words:

```bash
python rng-words.py 10
```

Prints 10 random words to the console.
