# Python Util Scripts

A collection of handy Python utility scripts that I need sometimes.

## Usage

Scripts may explain their usage using `py <script.py> --help` (assuming I implement it).
If not then inspect the code for its usage.

### Examples

To run the web image scraper:
```bash
python scrape.py https://example.com/
```

Downloaded images will appear in the `output` directory.
